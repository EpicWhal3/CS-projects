# File System Management CLI

Консольное приложение на C# для навигации и выполнения операций над файловой системой

## О проекте

В рамках лабораторной реализовано приложение для взаимодействия с файловой системой через консольные команды.

Поддерживаются операции:

- подключение к файловой системе;
- переход по директориям;
- просмотр дерева каталогов;
- просмотр содержимого файлов;
- копирование файлов;
- перемещение файлов;
- удаление файлов;
- переименование файлов;
- отключение от файловой системы.

Приложение спроектировано так, чтобы логика команд, вывод и работа с файловой системой были разделены и не зависели напрямую друг от друга.

## Поддерживаемые команды

- `connect [Address] [-m Mode]`
- `disconnect`
- `tree goto [Path]`
- `tree list {-d Depth}`
- `file show [Path] {-m Mode}`
- `file move [SourcePath] [DestinationPath]`
- `file copy [SourcePath] [DestinationPath]`
- `file delete [Path]`
- `file rename [Path] [Name]`

## Что реализовано

- консольный механизм работы с приложением;
- разбор команд с аргументами и флагами;
- навигация по абсолютным и относительным путям;
- просмотр дерева каталогов с настраиваемой глубиной;
- просмотр содержимого файлов;
- операции перемещения, копирования, удаления и переименования;
- переключение между файловыми системами/точками подключения;
- параметризуемый вывод дерева;
- корректная обработка конфликтов имён;
- архитектура, допускающая расширение новых режимов вывода и новых реализаций файловой системы;
- тесты для парсера команд.

## Архитектурные особенности

Проект построен с упором на слабую связанность компонентов.

В реализации отражены следующие идеи:

- обработка команд отделена от консоли;
- логика работы с файловой системой не зависит от конкретного интерфейса ввода/вывода;
- вывод дерева каталогов не привязан к консоли;
- логика отображения содержимого файла также выделена отдельно;
- структура команд позволяет расширять набор параметров и режимов работы без переписывания бизнес-логики;
- используется абстракция файловой системы, а не прямые вызовы только локальной реализации в прикладной логике.

## Особенности вывода

Для дерева каталогов предусмотрены:

- ограничение глубины выборки;
- древовидное отображение;
- настройка символов файлов, папок и отступов.

## Тестирование

Проект включает тестирование парсера команд

Проверяются:

- корректное распознавание типа команды;
- корректное извлечение аргументов;
- корректная обработка флагов и параметров.
