# C# Study Projects

Набор учебных проектов на C#

## Содержимое

### Educational Program Builder

Модель конструктора образовательных программ с доменными сущностями, авторством, копированием объектов на основе существующих и in-memory репозиториями.

**Что демонстрирует проект:**

- проектирование предметной области;
- применение SOLID, GRASP и порождающих паттернов;
- работу с инвариантами бизнес-логики;
- покрытие ключевых сценариев unit-тестами.

### Corporate Messaging System

Система распределения сообщений с несколькими типами адресатов: пользователь, мессенджер, дисплей и группа адресатов. Поддержаны фильтрация по важности, логгирование и отслеживание статуса сообщений.

**Что демонстрирует проект:**

- построение расширяемой объектной модели;
- применение структурных паттернов;
- изоляцию внешних интеграций от доменной логики;
- разработку тестируемых компонентов с использованием mock-объектов.

### File System Management CLI

Консольное приложение для навигации и операций с файловой системой: просмотр дерева каталогов, чтение файлов, копирование, перемещение, удаление и переименование.

**Что демонстрирует проект:**

- разработку CLI-приложений с парсингом команд и флагов;
- разделение логики команд, вывода и файловой системы;
- проектирование расширяемой архитектуры без жёсткой привязки к консоли;
- тестирование парсера команд.

### ATM System

Система банкомата с интерактивным консольным интерфейсом, разделением пользовательского и административного режимов, хранением данных в PostgreSQL и акцентом на чистую архитектуру приложения.

**Что демонстрирует проект:**

- построение многослойной архитектуры;
- отделение бизнес-логики от инфраструктуры;
- работу с PostgreSQL без ORM;
- тестирование сервисного слоя через mock-репозитории.

## Ключевые навыки, отражённые в репозитории

- C# / .NET
- ООП, SOLID, GRASP
- паттерны проектирования
- unit-тестирование
- mocking
- консольные приложения
- работа с файловой системой
- PostgreSQL
- проектирование слоистой и расширяемой архитектуры

## Структура

Ветки репозитория соответствуют отдельным лабораторным работам.  
Каждая лабораторная оформлена как самостоятельный учебный проект со своей предметной областью, архитектурой и набором тестов.
